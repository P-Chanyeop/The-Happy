# 더해피 발주 자동분류 프로그램 설계

## 1. 개요

더해피는 쇼핑몰 주문을 받아 여러 외부 납품업체에 발주를 넣는 업무를 수행한다.
현재 수작업으로 하고 있는 아래 과정을 자동화하는 프로그램을 개발한다.

### 현재 수작업 프로세스

1. 주문이 들어오면 엑셀파일(처음.xlsx)을 받음
2. 엑셀에서 수취인이름, 수취인주소, 수취인전화번호1, 상품명/옵션이름, 상품수량, 배송메세지를 뽑아냄
3. 상품마다 납품업체가 다르므로, 상품별로 해당 업체를 분류
4. 각 업체의 구글시트에 해당 주문 데이터를 수기로 복사/붙여넣기
5. 이 작업을 매일 반복 → 주문량 증가로 자동화 필요

## 2. 프로그램 핵심 기능

### 2-1. 엑셀 업로드 → 주문 데이터 자동 파싱

- 처음.xlsx 파일을 업로드하면 필요한 컬럼만 자동 추출
- 추출 대상: 수취인이름, 수취인주소, 수취인전화번호1, (상품명+옵션이름), 상품수량, 배송메세지

### 2-2. 상품명/옵션이름 합산 처리

- 엑셀의 **상품명** 열과 **옵션이름** 열은 합쳐서 매칭에 사용
- 이유: 어떤 고객(쇼핑몰)은 상품명으로 표시하고, 다른 고객은 옵션이름으로 표시하기 때문
- 매칭 시 옵션이름 우선, 없으면 상품명 사용

### 2-3. 상품-업체 매칭

- 매칭 테이블(matching_config.json)을 참조하여 각 주문의 상품을 해당 업체에 자동 분류
- 매칭 키워드 = 엑셀의 상품명 또는 옵션이름
- 매칭 결과 = 해당 업체 + 구글시트에 입력할 품목명

### 2-4. 부반장제어 엑셀 임시저장 (동시작업 충돌 방지)

- 구글시트는 여러 팀에서 동시다발적으로 발주를 넣음
- 다른 팀이 발주 넣는 도중에 프로그램이 작동하면 오류 발생 가능
- **따라서 부반장제어 엑셀파일에 먼저 임시저장 후, 한번에 구글시트에 밀어넣는 방식**

### 2-5. 구글시트 일괄 입력

- 부반장제어 엑셀에서 업체별로 분류된 데이터를 해당 업체의 구글시트 탭에 일괄 입력
- 입력 시 기존 데이터 아래에 이어서 추가

### 2-6. 미매칭 상품 알림 (누락 방지)

- 매칭 테이블에 등록되지 않은 상품(신규 상품 등)은 별도 표시
- 주문 누락, 오배송 방지를 위해 **매칭되지 않은(구글시트로 옮겨지지 않은) 상품을 반드시 알 수 있어야 함**
- 미매칭 항목은 빨간색 하이라이트 + 경고 메시지

### 2-7. 발주 대상/비대상 구분

- 발주 프로그램에 들어가는 품목과 들어가지 않는 품목이 있음
- 모든 품목을 구분해내야 함
- 의도적으로 제외하는 품목(비대상)과 매칭이 안 된 품목(미등록)을 구분하여 표시

## 3. 데이터 흐름

```
[주문 엑셀(처음.xlsx) 업로드]
       ↓
[데이터 파싱] - 수취인이름, 주소, 전화번호, 상품명+옵션이름(합산), 수량, 배송메세지
       ↓
[상품-업체 매칭] - matching_config.json 참조
       ↓
[매칭 결과 미리보기] - 매칭 성공(초록) / 미매칭(빨강) / 비대상(회색) 구분 표시
       ↓
[부반장제어 엑셀에 임시저장] - 동시작업 충돌 방지
       ↓
[구글시트 업체별 탭에 일괄 입력]
       ↓
[미매칭 상품 최종 확인] - 누락 방지
```

## 4. 입력 엑셀 구조 (처음.xlsx)

| 컬럼인덱스 | 내용 |
|-----------|------|
| 0 | 주문일시 |
| 1 | 수취인이름 |
| 2 | 수취인주소 |
| 3 | 수취인전화번호1 |
| 4 | 상품명 |
| 5 | 옵션이름 |
| 6 | 상품수량 |
| 7 | 수취인 우편번호 |
| 8 | 배송메세지 |

- **상품명(4)과 옵션이름(5)은 합산하여 매칭에 사용**
- 컬럼 인덱스는 설정에서 변경 가능 (엑셀 양식이 바뀔 수 있으므로)

## 5. 구글시트 출력 구조 (업체별 공통)

| 컬럼 | 내용 | 매핑 소스 |
|------|------|----------|
| A | 상호 | ← 수취인이름 |
| B | 주소 | ← 수취인주소 |
| C | 전화번호 | ← 수취인전화번호1 |
| D | 품목 | ← 매칭된 품목명 (정확한 문자열 필수) |
| E | 수량 | ← 상품수량 |
| F | 택배비 | ← 택배비 규칙에 따라 (아래 참조) |
| G | 송장번호/주문번호 | (비워둠) |
| H | 담당 | ← 고정값 "온라인" (설정에서 변경 가능) |
| I | 특이사항 | ← 배송메세지 |

### 품목명 정확성 주의사항

- 품목명은 **띄어쓰기, 글자 하나라도 틀리면 구글시트에서 문자를 인식 못하고 대금정산 시 오류 발생**
- 따라서 매칭 테이블에 등록된 품목명을 그대로 복사하여 입력해야 함

## 6. 택배비 규칙

택배비 포함 상품이 대다수이나, 미포함 상품의 경우 별도 처리가 필요하다.
택배비 규칙은 업체마다 다르며, 자주 바뀌므로 **프로그램에서 자유롭게 수정 가능해야 함**.

### 택배비 유형

| 유형 | 설명 | 구글시트 택배비란 처리 | 예시 |
|------|------|----------------------|------|
| free | 택배비 포함 (기본) | 비워둠 | 대부분 상품 |
| fixed | 고정 금액 | 해당 금액 입력 + **셀 빨간색 배경** | 강화대유 → 택배비란에 "1" 입력 |
| variable | 조건별 금액 | 해당 금액 입력 + **셀 빨간색 배경** | 무진농산, 한림푸드 → 2500, 3000, 4000, 4500, 5500 등 |
| logen_calc | 로젠택배 수량 계산 | 별도 행 추가 | 고향시래기 (아래 상세 설명) |

### 택배비 셀 빨간색 표시

- 택배비 미포함 상품(fixed, variable, logen_calc)의 경우, 구글시트 **택배비란에 빨간색 배경색** 표시
- 택배비란에 해당 금액 숫자 입력 (예: "1", "3000" 등)

### 로젠택배 계산 (고향시래기 특수 규칙)

- 고향시래기 업체의 경우, **상호명이 초록색으로 변하는 값**의 수량만큼 로젠택배를 계산
- 처리 방법:
  - 품목란에 "로젠택배" 입력
  - 수량란에 초록색으로 변한 상호의 수량만큼 입력
  - 별도 행으로 추가

## 9. 추가 요구사항

### 9-1. 매칭 자유 변경

- 엑셀파일 또는 구글시트의 상품명, 옵션이름, 품목은 **자주 바뀌기 때문에** 매칭을 자유롭게 바꿀 수 있어야 함
- 프로그램 GUI에서 매칭 추가/수정/삭제 가능

### 9-2. 업체 추가/제거

- 추가 업체가 생길 경우 업체를 추가/제거 가능해야 함
- 업체 추가 시 구글시트 URL, 탭 이름, 담당 고정값 등 설정

### 9-3. 담당 고정값 간편 수정

- 구글시트 "담당" 란에 입력하는 고정값 (기본: "온라인")
- "00", "000" 등으로 간단하게 수정 가능해야 함
- 설정 탭에서 변경

### 9-4. 택배비 규칙 수정

- 택배비 규칙은 그때마다 바뀌므로 프로그램에서 수정 가능해야 함

## 10. GUI 구성 (tkinter)

### 탭 구조

1. **발주 처리** - 엑셀 업로드 → 미리보기 → 매칭 실행 → 부반장제어 저장 → 구글시트 전송
2. **매칭 관리** - 업체 선택 → 상품키워드-품목명 매칭 CRUD + 택배비 규칙 설정
3. **업체 관리** - 업체 추가/수정/삭제 (구글시트URL, 탭이름, 담당고정값)
4. **설정** - 담당 고정값, 엑셀 컬럼 매핑 변경

### 발주 처리 탭 흐름

1. "엑셀 파일 열기" → 파일 선택
2. 데이터 로드 → 테이블에 미리보기 표시
3. "자동 매칭 실행" → 업체별 색상 구분 (매칭=초록, 미매칭=빨강, 비대상=회색)
4. 미매칭 항목 경고 표시
5. "부반장제어 저장" → 임시 엑셀 저장
6. "구글시트 전송" → 업체별 구글시트 탭에 일괄 입력

### 매칭 관리 탭

- 업체 선택 드롭다운
- 상품키워드 → 품목명 매칭 테이블 (추가/수정/삭제)
- 택배비 유형 선택 (free/fixed/variable/logen_calc)
- 택배비 금액 입력

### 업체 관리 탭

- 업체 목록 테이블
- 업체 추가/수정/삭제 폼 (업체ID, 업체명, 시트이름, 구글시트URL, 담당고정값)

### 설정 탭

- 담당 고정값 변경 (기본: "온라인")
- 엑셀 컬럼 인덱스 매핑 변경

## 11. 기술 스택

- Python 3 + tkinter (GUI)
- openpyxl (엑셀 읽기/쓰기)
- gspread + google-auth (구글시트 API)
- JSON (매칭 설정 저장 - matching_config.json)
- PyInstaller (exe 패키징)

## 12. 핵심 예외처리

- **미매칭 상품** → 빨간색 하이라이트 + 경고 메시지, 전송 전 반드시 확인
- **비대상 상품** → 회색 표시, 의도적 제외임을 명시
- **구글시트 동시접근** → 부반장제어 엑셀 임시저장 후 일괄 전송으로 충돌 방지
- **품목명 정확성** → 매칭 테이블에서 복사하여 입력, 수동 입력 최소화
- **택배비 빨간색 표시** → 택배비 미포함 상품은 구글시트 택배비 셀에 빨간색 배경 적용
- **엑셀 양식 변경** → 컬럼 매핑을 설정에서 변경 가능

## 13. 납품 조건

- 작업 기간: 5 영업일
- 금액: 60만원 (부가세 별도, 부가세 포함 66만원)
- 중간 보고 수시 진행
- 무상 유지보수: 개발 완료 후 1개월
- 이후 유지보수: 월 10만원 (선택)
