# 더해피 발주 자동분류 프로그램 설계

## 1. 개요

쇼핑몰 주문 엑셀(처음.xlsx) → 상품-업체 매칭 → 업체별 구글시트 자동 입력

## 2. 데이터 흐름

```
[주문 엑셀 업로드]
       ↓
[데이터 파싱] - 수취인이름, 주소, 전화번호, 상품명+옵션이름(합산), 수량, 배송메세지
       ↓
[상품-업체 매칭] - 매칭 테이블 참조
       ↓
[부반장제어 엑셀에 임시저장] - 동시작업 충돌 방지
       ↓
[구글시트 일괄 입력] - 업체별 시트에 한번에 밀어넣기
       ↓
[미매칭 상품 알림] - 누락 방지
```

## 3. 입력 엑셀 구조 (처음.xlsx)

| 컬럼 | 내용 |
|------|------|
| A | 주문일시 |
| B | 수취인이름 |
| C | 수취인주소 |
| D | 수취인전화번호1 |
| E | 상품명 |
| F | 옵션이름 |
| G | 상품수량 |
| H | 수취인 우편번호 |
| I | 배송메세지 |
| J | 결제일시 |
| K | 쇼핑몰 분류 |
| L | 운송장번호 |

- 상품명(E)과 옵션이름(F)은 합산하여 매칭에 사용 (둘 중 하나만 있는 경우도 있음)

## 4. 구글시트 출력 구조 (업체별 공통)

| 컬럼 | 내용 | 매핑 |
|------|------|------|
| A | 상호 | ← 수취인이름 |
| B | 주소 | ← 수취인주소 |
| C | 전화번호 | ← 수취인전화번호1 |
| D | 품목 | ← 매칭된 품목명 |
| E | 수량 | ← 상품수량 |
| F | 택배비 | ← 택배비 규칙에 따라 |
| G | 송장번호/주문번호 | (비워둠) |
| H | 담당 | ← 고정값 (예: "온라인", "00", "000") |
| I | 특이사항 | ← 배송메세지 |

## 5. 매칭 테이블 구조

JSON 파일(matching_config.json)로 관리:

```json
{
  "vendors": {
    "업체ID": {
      "name": "고향시래기",
      "sheet_name": "p1455고향시래기_우거지",
      "google_sheet_url": "https://docs.google.com/...",
      "google_sheet_gid": "76926240",
      "default_manager": "온라인",
      "products": {
        "농가염지우거지10키": { "item_name": "염지우거지10kg", "shipping_fee": null },
        "농가우거지10키": { "item_name": "우거지10kg", "shipping_fee": "로젠택배계산" }
      },
      "shipping_rules": {
        "type": "free|fixed|per_box|logen_calc",
        "amount": 0
      }
    }
  }
}
```

## 6. 택배비 규칙 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| free | 택배비 포함 | 대부분 상품 |
| fixed | 고정 금액 | 강화대유=1 (택배비란에 1 입력) |
| variable | 조건별 금액 | 무진농산=2500,3000,4000... |
| logen_calc | 로젠택배 수량계산 | 고향시래기: 초록색 상호 수량만큼 로젠택배 행 추가 |

## 7. GUI 구성 (tkinter)

### 메인 탭
1. **발주 처리** - 엑셀 업로드 → 미리보기 → 구글시트 전송
2. **매칭 관리** - 상품-업체 매칭 CRUD
3. **업체 관리** - 업체/구글시트URL 추가·수정·삭제
4. **설정** - 담당자 고정값, 택배비 규칙 등

### 발주 처리 탭 흐름
1. 엑셀 파일 선택
2. 데이터 로드 → 테이블에 미리보기 표시
3. 자동 매칭 실행 → 업체별로 색상 구분 표시
4. 미매칭 항목 빨간색 하이라이트 + 경고
5. "부반장제어 저장" 버튼 → 임시 엑셀 저장
6. "구글시트 전송" 버튼 → 업체별 구글시트에 일괄 입력

### 매칭 관리 탭
- 업체 선택 드롭다운
- 상품명/옵션이름 → 품목명 매칭 테이블 (추가/수정/삭제)
- 택배비 규칙 설정
- 검색/필터 기능

## 8. 기술 스택

- Python 3 + tkinter (GUI)
- openpyxl (엑셀 읽기/쓰기)
- gspread + google-auth (구글시트 API)
- JSON (매칭 설정 저장)

## 9. 핵심 예외처리

- 미매칭 상품 → 별도 목록으로 표시, 전송 전 확인
- 구글시트 동시접근 → 부반장제어 엑셀 임시저장 후 일괄 전송
- 엑셀 양식 변경 → 컬럼 매핑을 설정에서 변경 가능하게
