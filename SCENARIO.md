# 더해피 발주 자동분류 - 사용 시나리오

## 시나리오 1: 첫 세팅 (최초 1회)

### 1-1. 업체 등록
1. "업체 관리" 탭 클릭
2. 아래 정보 입력 후 "추가/수정" 클릭:
   - 업체ID: `p1455고향시래기_우거지`
   - 업체명: `우거지`
   - 시트이름: `p1455고향시래기_우거지`
   - 구글시트URL: (나중에 입력)
   - 담당고정값: `온라인`
3. 나머지 업체도 동일하게 등록 (이미 15개 등록되어 있으면 생략)

### 1-2. 상품-업체 매칭 등록
1. "매칭 관리" 탭 클릭
2. 업체 선택: "우거지" (고향시래기)
3. 아래 입력 후 "추가":
   - 상품키워드: `농가염지우거지10키`
   - 품목명: `염지우거지10kg`
   - 택배비유형: `logen_calc`
   - 택배비금액: (비워둠)
4. 다른 상품도 추가:
   - 상품키워드: `농가우거지10키` → 품목명: `우거지10kg` → 택배비유형: `logen_calc`
5. 업체 변경: "새우젓" (강화대유) 선택
6. 아래 입력 후 "추가":
   - 상품키워드: `국산추젓1키` → 품목명: `국산추젓1키` → 택배비유형: `fixed` → 택배비금액: `1`
7. 업체 변경: "김치" (한림푸드) 선택
8. 아래 입력 후 "추가":
   - 상품키워드: `포기김치10키` → 품목명: `포기김치10키` → 택배비유형: `variable` → 택배비금액: `3000`

### 1-3. 비대상 상품 등록 (발주 안 하는 품목)
1. "설정" 탭 클릭
2. 비대상 상품 입력란에 발주 제외할 상품키워드 입력 → "추가"
   - 예: 자체배송 상품, 단순 포장재 등

## 시나리오 2: 일일 발주 처리 (매일 반복)

> 이게 실제로 매일 하는 작업 흐름입니다.

### 2-1. 주문 엑셀 로드
1. 프로그램 실행 (`python main.py`)
2. "발주 처리" 탭에서 "엑셀 파일 열기" 클릭
3. 오늘 받은 주문 엑셀 파일 선택 (예: `260323처음.xlsx`)
4. 확인: 하단에 "총 N건 로드됨" 표시, 테이블에 주문 목록이 빨간색으로 표시

### 2-2. 자동 매칭 실행
1. "자동 매칭 실행" 클릭
2. 확인:
   - 매칭된 행 → 초록색, 매칭업체/품목 표시
   - 비대상 행 → 회색, "비대상" 표시
   - 미매칭 행 → 빨간색, "미매칭" 표시
3. 상태바 확인: "매칭: 15건 성공, 2건 비대상, 3건 미매칭" 같은 형태

### 2-3. 미매칭 상품 처리
미매칭이 있으면 경고 팝업이 뜸. 두 가지 경우:

**경우 A: 신규 상품 → 매칭 등록**
1. 미매칭 행의 상품명/옵션이름 확인
2. "매칭 관리" 탭으로 이동
3. 해당 업체 선택 → 상품키워드/품목명 입력 → "추가"
4. "발주 처리" 탭으로 돌아와서 "자동 매칭 실행" 다시 클릭
5. 해당 행이 초록색으로 변하면 성공

**경우 B: 발주 안 하는 상품 → 비대상 등록**
1. "설정" 탭으로 이동
2. 비대상 상품에 해당 키워드 추가
3. "발주 처리" 탭으로 돌아와서 "자동 매칭 실행" 다시 클릭
4. 해당 행이 회색으로 변하면 성공

### 2-4. 부반장제어 엑셀 저장
1. 모든 미매칭이 해결되었는지 확인 (빨간색 행이 없어야 함)
2. "부반장제어 저장" 클릭
3. 저장 위치/파일명 지정 → 저장
4. 저장된 엑셀 파일을 열어서 확인:
   - 업체별 시트(탭)로 분류되어 있는지
   - 품목명이 정확한지 (띄어쓰기, 글자 하나라도 틀리면 안 됨)
   - 택배비가 있는 셀은 빨간색 배경인지
   - 고향시래기 업체에 "로젠택배" 행이 추가되어 있는지
   - 담당란에 "온라인"이 들어가 있는지

### 2-5. 구글시트 전송 (Google API 설정 완료 후)
1. "구글시트 전송" 클릭
2. 전송 완료 메시지 확인
3. 실제 구글시트 열어서 데이터가 기존 행 아래에 추가되었는지 확인

## 시나리오 3: 매칭 정보 변경

> 상품명이 바뀌거나 택배비 규칙이 변경될 때

### 3-1. 기존 매칭 수정
1. "매칭 관리" 탭 → 업체 선택
2. 수정할 항목 클릭 → 하단 폼에 값이 채워짐
3. 품목명 또는 택배비 수정
4. "선택 수정" 클릭
5. 테이블에 변경 반영 확인

### 3-2. 매칭 삭제
1. 삭제할 항목 클릭
2. "선택 삭제" 클릭
3. 테이블에서 사라졌는지 확인

### 3-3. 택배비 규칙 변경
예: 강화대유 택배비가 1 → 2500으로 변경
1. "매칭 관리" 탭 → "새우젓"(강화대유) 선택
2. 해당 상품 클릭 → 택배비금액을 "2500"으로 수정
3. "선택 수정" 클릭

---

## 시나리오 4: 신규 업체 추가

> 새로운 납품업체가 생겼을 때

1. "업체 관리" 탭 → 업체ID/업체명/시트이름 입력 → "추가/수정"
2. "매칭 관리" 탭 → 새 업체 선택 → 상품 매칭 등록
3. 다음 발주부터 해당 업체로 자동 분류됨

---

## 시나리오 5: 담당 고정값 변경

> "온라인" 대신 다른 값을 넣어야 할 때

1. "설정" 탭 → 담당 고정값을 "00" 또는 "000"으로 변경 → "저장"
2. 이후 부반장제어 저장/구글시트 전송 시 변경된 값이 적용됨
