# 더해피 발주 자동분류 프로그램 테스트 체크리스트

## 1단계: 프로그램 실행

- [X] `python main.py` 실행 시 오류 없이 창이 뜨는지
- [X] 4개 탭 (발주 처리, 매칭 관리, 업체 관리, 설정) 모두 보이는지
- [X] 창 크기 조절 시 테이블이 정상적으로 리사이즈 되는지

## 2단계: 설정 탭

- [X] 담당 고정값이 "온라인"으로 표시되는지
- [X] 담당 고정값을 "00"으로 변경 → 저장 → 프로그램 재시작 후 유지되는지
- [X] 엑셀 컬럼 매핑 값이 0~8까지 정상 표시되는지
- [X] 컬럼 매핑 변경 → 저장 → matching_config.json에 반영되는지
- [X] 비대상 상품 추가: 키워드 입력 → "추가" → 리스트에 나타나는지
- [X] 비대상 상품 삭제: 항목 선택 → "선택 삭제" → 리스트에서 사라지는지
- [X] 비대상 상품 추가/삭제 후 matching_config.json에 excluded_products 반영되는지
- [X] 같은 키워드 중복 추가 시 중복 등록 안 되는지

## 3단계: 업체 관리 탭

- [X] 기존 15개 업체가 테이블에 모두 표시되는지
- [X] 업체 선택 시 하단 폼에 정보가 자동 채워지는지
- [X] 새 업체 추가: 업체ID/업체명/시트이름 입력 → "추가/수정" → 테이블에 나타나는지
- [X] 기존 업체 수정: 선택 → 업체명 변경 → "추가/수정" → 변경 반영되는지
- [X] 업체 삭제: 선택 → "선택 삭제" → 확인 팝업 → 삭제되는지
- [X] 구글시트URL 입력 후 저장 → json에 반영되는지
- [X] 담당고정값 비워두면 기본값("온라인")으로 저장되는지

## 4단계: 매칭 관리 탭

- [X] 업체 드롭다운에 모든 업체가 나오는지
- [X] 업체 선택 시 해당 업체의 매칭 목록이 표시되는지 (현재 비어있을 것)
- [X] 매칭 추가: 업체 선택 → 상품키워드/품목명 입력 → 택배비유형 "free" → "추가" → 테이블에 나타나는지
- [ ] 매칭 추가: 택배비유형 "fixed", 금액 "1" → 추가 → 정상 저장되는지
- [ ] 매칭 추가: 택배비유형 "variable", 금액 "3000" → 추가 → 정상 저장되는지
- [ ] 매칭 추가: 택배비유형 "logen_calc" → 추가 → 정상 저장되는지
- [X] 중복 키워드 추가 시 경고 메시지 나오는지
- [X] 매칭 선택 수정: 항목 선택 → 폼에 값 채워지는지 → 수정 후 "선택 수정" → 반영되는지
- [ ] 매칭 선택 수정: 키워드 자체를 변경 → "선택 수정" → 기존 키워드 삭제되고 새 키워드로 저장되는지
- [X] 매칭 삭제: 항목 선택 → "선택 삭제" → 삭제되는지
- [X] 업체를 선택 안 하고 추가 시 경고 나오는지
- [X] 키워드/품목명 비워두고 추가 시 경고 나오는지
- [ ] "새로고침" 버튼 동작하는지

## 5단계: 발주 처리 탭 - 엑셀 로드

- [X] "엑셀 파일 열기" → 260323처음.xlsx 선택 → 데이터가 테이블에 표시되는지
- [X] 파일명이 상단에 표시되는지
- [X] 하단 상태바에 "총 N건 로드됨" 표시되는지
- [X] 각 행의 주문일시, 수취인, 주소, 전화번호, 상품명, 옵션이름, 수량, 배송메세지가 정상인지
- [X] 빈 행은 건너뛰는지
- [X] 로드 직후에는 모든 행이 빨간색(미매칭)인지

## 6단계: 발주 처리 탭 - 자동 매칭

> 테스트 전 준비: 매칭 관리 탭에서 260323처음.xlsx에 있는 상품 몇 개를 매칭 등록해둘 것

- [X] "자동 매칭 실행" → 매칭된 행은 초록색으로 변하는지
- [X] 미매칭 행은 빨간색 유지되는지
- [X] 비대상 상품(excluded_products에 등록한 것)은 회색으로 표시되는지
- [X] 매칭업체 컬럼에 업체명이 표시되는지
- [ ] 비대상 행은 매칭업체 컬럼에 "비대상"으로 표시되는지
- [ ] 미매칭 행은 매칭업체 컬럼에 "미매칭"으로 표시되는지
- [ ] 품목 컬럼에 매칭된 품목명이 정확히 표시되는지
- [ ] 상태바에 "매칭: N건 성공, N건 비대상, N건 미매칭" 표시되는지
- [ ] 미매칭이 있으면 경고 팝업이 뜨는지
- [ ] 옵션이름이 있는 주문은 옵션이름으로 매칭되는지
- [ ] 옵션이름이 비어있는 주문은 상품명으로 매칭되는지
- [ ] 엑셀 로드 안 하고 매칭 실행 시 경고 나오는지

## 7단계: 부반장제어 저장

- [ ] "부반장제어 저장" → 파일 저장 다이얼로그 뜨는지
- [ ] 저장된 xlsx 파일 확인:
  - [ ] 업체별로 시트(탭)가 나뉘어 있는지
  - [ ] 각 시트에 헤더(상호/주소/전화번호/품목/수량/택배비/송장번호/담당/특이사항)가 있는지
  - [ ] 데이터가 올바른 업체 시트에 들어갔는지
  - [ ] 담당 컬럼에 고정값("온라인" 또는 설정한 값)이 들어갔는지
  - [ ] 배송메세지가 특이사항에 들어갔는지
  - [ ] 비대상 상품은 제외되었는지
  - [ ] 미매칭 상품은 "미매칭" 시트에 모여있는지
- [ ] 택배비 확인:
  - [ ] free 상품 → 택배비 칸 비어있는지
  - [ ] fixed 상품 → 택배비 칸에 금액 + 셀 배경 빨간색인지
  - [ ] variable 상품 → 택배비 칸에 금액 + 셀 배경 빨간색인지
  - [ ] logen_calc 상품 → 원래 행 아래에 "로젠택배" 행이 추가되었는지
  - [ ] logen_calc 상품 → 로젠택배 행의 상호 셀이 초록색 배경인지
  - [ ] logen_calc 상품 → 로젠택배 행의 수량이 원래 주문 수량과 같은지

## 8단계: 구글시트 전송

- [ ] credentials.json 없이 전송 시 → 오류 메시지에 파일 경로 안내가 나오는지
- [ ] gspread 미설치 시 → pip install 안내 메시지 나오는지
- [ ] 매칭된 주문 없이 전송 시 → "전송할 매칭된 주문이 없습니다" 경고 나오는지
- [ ] (Google API 설정 후) 업체 구글시트URL 미입력 업체는 건너뛰는지
- [ ] (Google API 설정 후) 실제 구글시트에 데이터가 기존 행 아래에 추가되는지
- [ ] (Google API 설정 후) 로젠택배 행도 구글시트에 추가되는지

## 9단계: 데이터 영속성

- [ ] 프로그램 종료 후 재시작 → 업체 목록 유지되는지
- [ ] 프로그램 종료 후 재시작 → 매칭 데이터 유지되는지
- [ ] 프로그램 종료 후 재시작 → 비대상 상품 목록 유지되는지
- [ ] 프로그램 종료 후 재시작 → 설정값(담당, 컬럼매핑) 유지되는지
- [ ] matching_config.json을 직접 열어서 데이터가 정상인지

## 10단계: 예외/엣지 케이스

- [ ] 빈 엑셀 파일 로드 시 오류 없이 "총 0건" 표시되는지
- [ ] 엑셀 파일 선택 취소 시 아무 일도 안 일어나는지
- [ ] 부반장제어 저장 취소 시 아무 일도 안 일어나는지
- [ ] 주문 없이 부반장제어 저장 시 아무 일도 안 일어나는지
- [ ] 상품명/옵션이름 둘 다 비어있는 행이 있을 때 매칭이 오류 없이 처리되는지
- [ ] 수량이 비어있는 행 → 0으로 처리되는지
